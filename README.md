CAEN Log Viewer is a lightweight Python-based GUI application for visualizing and analyzing log files from CAEN high-voltage power supplies. Built with PyQt5 and Plotly, the tool is designed to help detector physicists and engineers monitor and debug HV systems with ease. It was developed by @Luca Moleri with ChatGPT
Key Features:

   - 📁 Load native CAEN .log or .txt files directly
   - 📊 Plot voltage, current, or other parameters over time
   - 🎛️ Filter by channel and select multiple parameters
   - 🪟 Each channel is displayed in its own plot canvas, with toggleable visibility

The tool is modular and extensible, intended for rapid diagnostics and long-term monitoring of HV system behavior in experimental setups.
